About This Project

This project is an Image Forgery Detection System designed to classify whether an uploaded image is fake or real. We use an ELA-based preprocessing technique and combine RGB and LAB edge detection methods to extract meaningful tampering features. A deep learning model trained on the CASIA v2 dataset performs the classification.

The aim of this project is to assist in detecting forged or manipulated images in digital media, which is important for journalism, forensics, and legal investigations.

What We Have Done

  • Preprocessed images using Error Level Analysis (ELA)
  • Extracted RGB and LAB color edge features
  • Trained a CNN model using TensorFlow and Keras
  • Built a web interface using Flask for uploading and predicting images
  • Added visual output of processed images with edge detection overlay
  • Designed a responsive and interactive UI
